# Meteor Idle — WebGL build

This branch hosts the built WebGL player for **Meteor Idle**, served via GitHub Pages at:

<https://muwamath.github.io/Meteor-Idle/>

The source code lives on the `main` branch. Deploys are pushed from `tools/deploy-webgl.sh` in the main branch after each branch is promoted to `main`.

Do not edit files here by hand — they are regenerated from every Unity build.
